# backToThe1992

modify github profile contribution history to 1992

## Usage

Create a new repo on GitHub.

Generate a personal access token on GitHub.

```sh
pnpm i

./backToThe1992.sh -u=username -y=1992 --repo=repo_name --token=access_token
```
